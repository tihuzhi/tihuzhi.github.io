标题用法

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题

注意：

  • 最后一个#​ 字符与标题中间留一个空格
  • 标题应该置于行首，如果放入表格中可能无法正确解析

字体用法

*这是斜体*​ : 这是斜体

_这是斜体_​ : 这是斜体

**这是粗体**​ : 这是粗体

__这是粗体__​ : 这是粗体

***这是粗斜体***​ : 这是粗斜体

___这是粗斜体___​ : 这是粗斜体

快捷键：
加粗 Ctrl+B
斜体 Ctrl+I

换行用法

Markdown 换行的方式有很多种:

  • 直接在一句话后敲两个空格
  • 两句话之间加一个空行
  • 如果你在编辑的时候，想让一行文字在显示的时候换行，就在中间加<br/>​
  • 代码：
    这是一<br>段话
    演示：

这是
一段话
‍

引用用法

Markdown 中引用通过符号 >​ 来实现。>​ 符号后的空格，可有可无。
在引用的区块内，允许换行存在，换行并不会终止引用的区块。如果要结束引用，需要一行空白行，来结束引用的区块。
代码一：>这是一个引用​

效果一：

这是一个引用

代码二：

>这是一个引用​

>>这是一个引用的引用​

>>>这是一个引用的引用的引用​

效果二：

这是一个引用：

这是一个引用的引用

这是一个引用的引用的引用

‍

链接用法

Markdown 中插入链接的使用方式是：

代码：

链接名称<链接地址> 即是：[这是是我的主页] (https://www.zhenliriji.tech/)

或者<https://www.zhenliriji.tech/>

效果：

我的博客

https://www.zhenliriji.tech/
